This web app has been developed with Spring Framework (Java EE) on the back end, using Spring Boot, Spring MVC, Spring Data, Spring Security, JaperReports, Hibernate, PostgreSQL, Java Mail and technologies such as HTML5, Thymeleaf, JQuery and Bootstrap on the front end.

Perform requirements analysis, development, maintenance and testing of FPI – (e-Invoicing), cloud-based app (Azure) for companies obligated for DIAN, the National Colombian Tax Authority (Dirección de Impuestos y Aduanas Nacionales), to use their platform for sending and receiving documents, and in that way comply with the latest tax reform (Law 1819 of 2016). This application consists of the following three main components:

  • Web Portal: built using Java EE on the back end (Spring Boot, Spring MVC, Spring Security, Spring Data, JaperReports, Hibernate); DBMS such as MySQL and SQLServer, and on the front end we use Thymeleaf, JQuery, AJAX, Bootstrap and Jqgrid.
  • RESTful API: built with Spring REST and Swagger UI for documentation.
  • SOAP client: for sending and receiving documents and JAXB for XML manipulation.

In order to work together, these three FPI components are integrated via EIP (Enterprise Integration Patterns) Java implementation known as Apache Camel and Active MQ as messaging system.
